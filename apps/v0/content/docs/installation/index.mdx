# Installation

ImBIOS UI components can be installed using two methods, each with its own advantages. Choose the method that best fits your workflow.

## Installation Methods

| Method          | Command                       | Best For                                  |
| --------------- | ----------------------------- | ----------------------------------------- |
| **NPM Package** | `pnpm add @imbios/ui`         | Managed updates, quick setup              |
| **shadcn CLI**  | `npx shadcn@latest add "URL"` | Full customization, source code ownership |

### NPM Package (Recommended for Quick Setup)

Install as a dependency with automatic updates via your package manager.

```bash
pnpm add @imbios/ui
```

**Pros:**

- Quick setup (under 5 minutes)
- Automatic updates with `pnpm update`
- Smaller bundle size (tree-shaking)

**Cons:**

- Limited customization (prop-based only)
- External dependency

### shadcn CLI (Recommended for Customization)

Copy the component source code directly into your project. You own the code and can customize everything.

```bash
npx shadcn@latest add "https://ui.imbios.dev/r/credential-login"
```

**Pros:**

- Full control over styling and behavior
- No external dependencies to update
- Easy to modify for specific needs

**Cons:**

- Manual updates when upstream changes
- Larger initial setup

## Prerequisites

Before installing, ensure you have:

- **Node.js 18+**
- **React 18+** or **React 19+**
- **Tailwind CSS** configured
- **better-auth 1.4+** installed and configured

## Quick Start

Already have better-auth configured? Jump straight to installation:

```bash
# Option 1: NPM Package
pnpm add @imbios/ui

# Option 2: shadcn CLI
npx shadcn@latest add "https://ui.imbios.dev/r/credential-login"
```

Then import and use:

```tsx
import { CredentialLoginForm } from "@imbios/ui";
// or: import { CredentialLoginForm } from '@/components/auth/credential-login-form'

<CredentialLoginForm
  authMethod="email"
  authClient={authClient}
  showRememberMe={true}
  onSuccess={() => router.push("/dashboard")}
/>;
```
