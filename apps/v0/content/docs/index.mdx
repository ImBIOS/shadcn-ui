## Installation Methods

<Tabs items={['shadcn CLI', 'npm Package']}>
  <Tab value="shadcn CLI">
    ```bash
    npx shadcn@latest add "https://ui.imbios.dev/r/credential-login"
    ```

    - Full source code ownership
    - Complete customization freedom
    - Manual updates (re-run CLI)
  </Tab>
  <Tab value="npm Package">
    ```bash
    pnpm add @imbios/ui
    ```

    - Managed updates via npm
    - Prop-based configuration
    - Smaller maintenance overhead
  </Tab>
</Tabs>

## Quick Example

```tsx
import { CredentialLoginForm } from '@imbios/ui'
import { authClient } from '@/lib/auth-client'

export function SignInPage() {
  return (
    <CredentialLoginForm
      authMethod="email"
      authClient={authClient}
      showRememberMe
      showForgotPassword
      onSuccess={() => window.location.href = '/dashboard'}
    />
  )
}
```
